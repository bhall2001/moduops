# Introduction #

From the LiveCode wiki page…


The LiveCode programming language (formerly the "Revolution" programming language) is both an open-source and proprietary cross-platform rapid application development language inspired by HyperCard's programming language HyperTalk.

# Details #

LiveCode runs on iOS, Android, Mac OS X, Windows 95 through Windows 7, and several variations of Unix, including Linux, Solaris, and BSD. It can be used for mobile, desktop and server/CGI applications.
